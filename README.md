# journalism-tools

Data journalism scripts: GA4 audience analytics, PushEngage notification import, Mongabay quote extractor, image batch downloader

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env   # then fill in your keys
```

## Environment Variables

```
  GA4_PROPERTY_ID=your_ga4_property_id_here
  GA4_SERVICE_ACCOUNT_JSON=your_ga4_service_account_json_here
  ONESIGNAL_REST_API_KEY=your_onesignal_rest_api_key_here
```
